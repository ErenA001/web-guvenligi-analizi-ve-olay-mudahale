# Gün 5 - Basic Attack Detection

Bugün log analyzer scriptine basit saldırı tespiti eklendi.

Önceki gün IP bazlı trafik analizi yapılmıştı. Bugün bunun üzerine 401 Unauthorized ve 403 Forbidden durum kodları ayrıca kontrol edildi.

## Yapılanlar

- 401 status code değerleri IP bazlı sayıldı.
- 403 status code değerleri IP bazlı sayıldı.
- `/login` üzerindeki başarısız denemeler kontrol edildi.
- Bir IP birden fazla 401 veya 403 alıyorsa suspicious olarak gösterildi.
- Terminal çıktısına şüpheli IP listesi eklendi.

## Not

Bugün sadece temel attack detection mantığı eklendi. Brute force detection, severity scoring ve incident response sonraki günlere bırakıldı.

## Sonuç

Script başarıyla çalıştı ve örnek log dosyasında şüpheli IP adresleri tespit edildi.
