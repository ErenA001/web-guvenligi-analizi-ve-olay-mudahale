# Web Güvenliği Analizi ve Olay Müdahale

Bu repository, staj süreci boyunca geliştirilecek olan **Web Güvenliği Analizi ve Olay Müdahale** projesini içermektedir. Proje kapsamında web logları ve güvenlik olayları analiz edilerek şüpheli aktivitelerin tespit edilmesi ve bu olaylara yönelik müdahale önerilerinin üretilmesi hedeflenmektedir.

Repository, geliştirme süreci boyunca düzenli olarak güncellenecek; yapılan çalışmalar günlük raporlar, kod geliştirmeleri ve proje dokümantasyonu ile adım adım kayıt altına alınacaktır.
